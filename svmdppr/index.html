SVM-QoS-PRR

A research with the purpose on designing and simulating an approch to guarantee the Quality of Service (QoS) of a cloud services, focusing mainly for Software-as-a-Service (SaaS)

SVM-QoS-PRR

stands for

Support Vector Machine Based Cloud Quality Of Service Violation Prediction with Rule-Based Rectification

SVM-QoS-PRR approach contains 2 major process in sequential order:

  1. Cloud QoS violation detection and prediction process, which consist of a multi-class classification model trained with historical QoS dataset from cloud services using supervised machine learning algorithms, Support Vector Machine (SVM) to classify into decision rule indicating cloud QoS violation in great accuracy.
  2. Cloud QoS violation prevention and rectification process, which are responsible to rectify cloud QoS violation by assign on either prevention task for deploying scalability or rectification task for deploying fault tolerance capabilities based on decision rule received from previous cloud QoS violation detection and prediction process.
